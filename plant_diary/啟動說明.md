# 植物日記 - 快速啟動指南

## 快速開始

### 1. 安裝依賴套件

在 PowerShell 中執行（**推薦使用此命令**）：

```powershell
python -m pip install --only-binary :all: -r plant_diary_requirements.txt
```

> **提示**：使用 `--only-binary :all:` 參數可以避免編譯問題，特別是在 Python 3.14 等新版本上。

如果上面的命令失敗，可以查看 `安裝依賴說明.md` 文件獲取詳細的安裝指南。

### 2. 啟動應用程式

**方法一：使用啟動腳本（推薦）**

在項目根目錄執行：

```powershell
python run_plant_diary.py
```

**方法二：直接運行主程式**

```powershell
cd plant_diary
python main.py
```

或者：

```powershell
python plant_diary/main.py
```

## 首次使用

1. **添加第一個植物（推薦使用 OCR 識別）**
   - 點擊左側的「+ 添加新植物」按鈕
   - 點擊「📷 從照片識別」按鈕
   - 選擇植物花牌照片（確保照片清晰，文字可見）
   - 等待系統識別（通常幾秒鐘）
   - 系統會自動填充識別到的中文名稱和學名
   - 檢查識別結果，如有需要可手動修改
   - 添加備註（可選）
   - 點擊「保存」
   
   或者手動輸入：
   - 直接輸入中文名稱（必填）和學名（可選）
   - 點擊「保存」

2. **上傳第一張照片**
   - 選擇剛才添加的植物
   - 點擊「上傳照片」按鈕
   - 選擇照片文件
   - 照片會自動保存並進行 AI 分析

3. **查看 AI 分析結果**
   - 上傳照片後，點擊照片下方的「查看詳情和AI分析」按鈕
   - 在「AI 分析」標籤頁查看結果

## 設置 AI 功能（可選）

要使用完整的 AI 分析功能，需要設置 OpenAI API 密鑰：

1. 前往 https://platform.openai.com 註冊帳號
2. 創建 API 密鑰
3. 設置環境變數：

**Windows PowerShell:**
```powershell
$env:OPENAI_API_KEY="sk-your-api-key-here"
```

**Windows CMD:**
```cmd
set OPENAI_API_KEY=sk-your-api-key-here
```

> **提示**：如果不設置 API 密鑰，應用程式仍然可以使用，但 AI 分析功能將使用基礎模式。

## 常見問題

### Q: 應用程式無法啟動？
A: 確認已安裝所有依賴套件，特別是 Pillow。

### Q: 照片無法顯示？
A: 檢查照片文件格式是否支持（JPG, PNG, BMP, GIF），確認文件沒有損壞。

### Q: AI 分析不工作？
A: 
- 確認已設置 OPENAI_API_KEY 環境變數
- 確認網絡連接正常
- 檢查 API 密鑰是否有效且有餘額

### Q: 數據存儲在哪裡？
A: 
- 數據庫文件：`plant_diary.db`（在運行目錄）
- 照片文件：`plant_photos/` 目錄（在運行目錄）

## 功能提示

- **定期記錄**：建議每隔一個月左右為植物拍照一次，記錄成長歷程
- **詳細備註**：在添加照片時可以添加備註，記錄當時的觀察
- **AI 建議**：AI 分析可以幫助識別植物狀態，但建議結合自己的經驗判斷
