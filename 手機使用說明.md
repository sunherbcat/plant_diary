# 植物日記 - 手機使用說明

## 📱 現在可以在手機上使用！

我已經為您創建了 Web 版本，您現在可以在 Android 手機（或任何智能手機）的瀏覽器中使用「植物日記」了！

## 🚀 快速開始

### 第一步：安裝 Flask

在 PowerShell 中執行：

```powershell
python -m pip install Flask
```

### 第二步：啟動 Web 服務器

在項目目錄中執行：

```powershell
cd plant_diary_web
python app.py
```

或者雙擊 `啟動Web版.bat` 文件。

服務器啟動後會顯示：
```
 * Running on http://0.0.0.0:5000
```

### 第三步：從手機訪問

1. **確保手機和電腦連接在同一個 Wi-Fi 網絡**

2. **查看電腦的 IP 地址**：
   - 在 PowerShell 中執行：`ipconfig`
   - 查找「IPv4 地址」，通常是 `192.168.x.x` 格式
   - 例如：`192.168.1.100`

3. **在手機瀏覽器中訪問**：
   - 打開手機瀏覽器（Chrome、Safari 等）
   - 輸入：`http://電腦IP:5000`
   - 例如：`http://192.168.1.100:5000`

## ✨ 手機版功能

Web 版本包含所有核心功能：

- ✅ **添加植物**：支持手動輸入或從照片識別
- ✅ **OCR 識別**：拍攝植物花牌照片，自動識別中文名稱和學名
- ✅ **上傳照片**：直接用手機拍照並上傳
- ✅ **查看成長歷程**：查看所有照片的時間線
- ✅ **響應式設計**：完美適配手機屏幕

## 📸 手機拍照功能

Web 版本特別優化了手機拍照功能：

1. 點擊文件選擇按鈕時，會自動調用手機相機
2. 可以直接拍照並上傳
3. 照片會自動進行 AI 分析（如果設置了 OpenAI API 密鑰）

## 💡 使用技巧

### 方法一：直接拍照上傳

1. 選擇植物
2. 點擊「上傳照片」
3. 選擇「拍照」（會直接調用手機相機）
4. 拍照後自動上傳

### 方法二：OCR 識別植物花牌

1. 點擊「+ 添加新植物」
2. 點擊「📷 從照片識別」
3. 選擇「拍照」或「從相冊選擇」
4. 系統會自動識別並填充植物信息

## 🔒 安全提示

1. **僅在家庭網絡使用**：Web 服務器在本地網絡上運行，只適合在信任的網絡中使用
2. **防火牆設置**：如果無法訪問，可能需要允許防火牆通過端口 5000
3. **數據存儲**：所有數據仍然存儲在您的電腦上，不在手機上

## 🆘 常見問題

### Q: 無法從手機訪問？

**A:** 檢查以下幾點：

1. **確認手機和電腦在同一 Wi-Fi**：這是必須的
2. **確認 IP 地址正確**：使用 `ipconfig` 查看實際 IP（不是 localhost）
3. **檢查防火牆**：Windows 防火牆可能需要允許端口 5000
4. **確認服務器正在運行**：檢查電腦上的服務器是否正常啟動

### Q: 如何設置防火牆？

**A:** 在 Windows 中：

1. 打開「Windows 安全中心」
2. 點擊「防火牆和網絡保護」
3. 點擊「進階設置」
4. 創建新的「入站規則」，允許端口 5000

或者簡單方法：當 Windows 提示時，選擇「允許訪問」

### Q: 可以使用數據流量訪問嗎？

**A:** 不可以。Web 服務器只在本地網絡上運行，需要手機和電腦連接在同一個 Wi-Fi 網絡。

### Q: 可以同時使用桌面版和手機版嗎？

**A:** 可以！兩個版本使用相同的數據庫和照片文件，數據會自動同步。但要注意：
- 不要同時運行兩個桌面版
- 可以同時使用一個桌面版和一個 Web 版

### Q: 手機上的數據會同步到電腦嗎？

**A:** 是的！所有數據都存儲在電腦上，手機只是通過瀏覽器訪問。無論您從手機還是電腦操作，數據都會同步。
