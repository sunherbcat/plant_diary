# Python 安裝指南

## 問題
如果看到 "pip : 無法辨識" 或 "Python was not found" 的錯誤訊息，這表示您的系統還沒有安裝 Python。

## 解決方法：安裝 Python

### 方法一：從 Microsoft Store 安裝（推薦，最簡單）

1. 按 `Win` 鍵（Windows 鍵）
2. 搜尋 "Microsoft Store"
3. 在 Store 中搜尋 "Python 3.12" 或 "Python"
4. 點擊 "取得" 或 "安裝"
5. 等待安裝完成

### 方法二：從 Python 官網安裝（推薦，最新版本）

1. 開啟瀏覽器，前往：https://www.python.org/downloads/
2. 點擊 "Download Python 3.12.x"（會自動選擇最新版本）
3. 下載完成後，執行安裝程式
4. **重要**：在安裝時，務必勾選 **"Add Python to PATH"**（將 Python 加到系統路徑）
5. 點擊 "Install Now" 開始安裝

## 驗證安裝

安裝完成後，請重新開啟 PowerShell（或關閉後重新開啟），然後執行：

```powershell
python --version
```

如果顯示版本號（例如：`Python 3.12.0`），表示安裝成功！

## 安裝依賴套件

Python 安裝成功後，在 PowerShell 中執行以下命令來安裝所需的套件：

```powershell
python -m pip install -r requirements.txt
```

或者：

```powershell
pip install -r requirements.txt
```

## 如果還是無法使用 pip

如果 Python 已安裝但仍無法使用 pip，請嘗試：

```powershell
python -m pip install -r requirements.txt
```

這個命令使用 `python -m pip`，即使 pip 不在 PATH 中也能運作。

## 注意事項

- 如果目錄路徑包含中文字符（如「童話花園」），某些命令可能會有問題
- 如果遇到路徑問題，可以嘗試將專案資料夾移動到不含中文的路徑
- 或者使用英文路徑，例如：`C:\Users\User\Documents\FairyGarden`
